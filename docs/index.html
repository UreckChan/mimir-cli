Mimir CLI

Autonomous coding agent with 4 specialized roles that work in a loop. No API key required. Works in your terminal with zero setup.

Agente autónomo de código con 4 roles especializados que trabajan en ciclo. Sin API key. Funciona en tu terminal sin configuración.

The 4 RolesLos 4 Roles

ROLE_01

Architect

Plans strategy. Breaks down goals into steps.

Planea la estrategia. Desglosa objetivos en pasos.

ROLE_02

Builder

Writes code, runs commands, installs deps.

Escribe código, ejecuta comandos.

ROLE_03

Inspector

Verifies quality. Runs tests, checks code.

Verifica calidad. Ejecuta tests.

ROLE_04

Chronicler

Documents lessons. Generates playbooks.

Documenta lecciones. Genera playbooks.

SecuritySeguridad

API keys and credentials are automatically censored from logs and tool outputs before reaching the AI model. Three permission levels available.

Las API keys se censuran automáticamente de los logs antes de llegar al modelo de IA.

Alexandria Vault

Every lesson learned is stored in a persistent local vault with FTS5 search. Past solutions are recalled automatically when similar tasks appear.

Cada lección se guarda en una bóveda persistente con búsqueda FTS5.
